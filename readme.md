# rocketseat_ignite_ch02
- Challange 002 of the Ignite training by Rocketseat
- It's basically an Rest API that provides services to create a TO DO list.
- The full challange description can be found [here](https://www.notion.so/Desafio-02-Trabalhando-com-middlewares-4f89bf538c2e4ee291382b92bdc36790)
- Template used can be found [here](https://github.com/rocketseat-education/ignite-template-trabalhando-com-middlewares)

## Tech/framework used
- NodeJS
- ExpressJS
- Yarn
- Nodeman
- Jest

## Running
To run this sample use:

``` yarn dev ```

To test it use:

``` yarn test ```
